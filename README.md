## About AIMS

Asset Inventory Management System, this system will serve as an integrated platform for tracking, tagging, maintaining and reporting
physical assets throughout KESIA operations. 

## Functional Features
1. Asset Tagging and Registration
2. Assignment and Custody Tracking
3. Asset Scanner
4. License and Permit Management
5. Employee Management
6. Setup and Maintenance (Location, Sub-location, Asset Category)
7. Reports and Dashboard
